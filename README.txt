1) Unzip necessary driver for your OS.
2) Check chrome version you have. Put into browser command line: 'chrome://version'
    The provided chrome driver is used for chrome 77 version. If you use another one version refer to
    https://sites.google.com/a/chromium.org/chromedriver/downloads
    and download compatible version of chrome driver.

##Windows

2) Download JVM and do all activity refer to https://www.youtube.com/watch?v=74pE3kLerAU
3) open 'windows-script.cmd' file by notepad or any text-editor
4) check/change variables (ROOT_FOLDER_PATH, DRIVER_PATH)
5) change last parameters according your business requirements
6) execute your script-file: click on the 'windows-script.cmd'
